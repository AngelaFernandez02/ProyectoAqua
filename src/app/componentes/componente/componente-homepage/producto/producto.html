Descripción del Sistema de Captación de Lluvia

Nuestro sistema inteligente recolecta, almacena y monitorea el agua de lluvia usando sensores IoT y una app Android. Es una solución automatizada y ecológica que permite ahorrar agua y cuidar el medio ambiente.

¿Cómo funciona?

  • Detecta la lluvia y niveles de agua mediante sensores conectados a una placa IoT.
  • Los datos se envían a la nube vía Wi-Fi para ser visualizados en tiempo real.
  • Desde una app o interfaz web se puede monitorear y controlar el sistema.
